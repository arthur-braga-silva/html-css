Listas

Listas ordenadas

Para se fazer listas ordenadas se utiliza a tag ol para começar e li para sequência

  1. acordar
  2. tomar café
  3. ler um jornal
  4. tomar banho
  5. e escovar os dentes

Listas não ordenadas

Para se ter listas não ordenadas, se usa a tag ul e para a sequência também li

  • tomar banho
  • beber cafe
  • escovar os dentes
  • dormir
