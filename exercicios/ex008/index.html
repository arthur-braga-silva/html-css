Também podemos criar um texto marcado usando a tag mark

mas agora, para se ter marcado e com cor diferenciada kk, faça isso: olá mundo

Agora, para se ter um texto com a letra menor, utilize a tag: small. EX: Olhe o tamanho da importância do dinheiro

agora para riscar o texto, ou seja, aparecer como texto excluído, utiliza-se a tag: del. EX: Eu estou infeliz

Para se ter um texto com algo elevado (sobrescrito) se usa a tag: sup. EX: Faça a conta x2+b2+a2

Agora, para se ter um texto subescrito, se usa a tag: sub. EX: Olha o H20
